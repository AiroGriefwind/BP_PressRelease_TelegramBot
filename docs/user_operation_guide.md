# Telegram Bot 使用操作指南（用戶版）

本指南整理目前可用功能，供日常操作快速查閱。  
介面文字以 Bot 內實際按鈕為準（繁體）。

---

## 一、快速開始

1. 先把公關稿正文檔（非圖片）與附件傳給 Bot。  
2. 在聊天室 `@Bot` 喚起主界面。  
3. 依需求操作：`確認 / FB URL / 刪除 / ⚙️ 設定 / 🧾 Logs / ❓ 幫助 / 🔄 刷新 / 🛑 結束會話`。  

---

## 二、主界面按鈕說明

- `確認`：送出目前附件與設定。  
- `FB URL`：走 Facebook 連結傳送流程。  
- `刪除`：刪除單個或全部附件。  
- `⚙️ 設定`：調整類型、優先度、語言、傳送方式。  
- `🧾 Logs`：查看近期傳送記錄與失敗原因。  
- `❓ 幫助`：查看功能操作教學。  
- `🔄 刷新`：重新整理主界面。  
- `🛑 結束會話`：主動結束目前會話。  

---

## 三、功能操作教學

### 1) 傳送公關稿（附件上傳 → 確認傳送）

**目的**  
把公關稿正文與附件傳送到目標信箱，並完成本次會話。

**前置條件**  
已開啟主界面，或已先傳送附件後再 @Bot 喚起主界面。

**操作步驟**
1. 向 Bot 傳送公關稿正文檔（非圖片）與需要的附件/圖片。  
2. 如尚未出現主界面，可在傳送附件後 @Bot 喚起。  
3. 等待 Bot 提示「已添加…當前累計…」。  
4. 在主界面確認「附件列表」是否完整。  
5. 如需補充或刪除附件，先完成附件調整。  
6. 點擊「確認」開始傳送。  
7. 觀察進度提示，等待「傳送完成」。  
8. 若提示「會話結束」，表示本次已完成。  

**注意事項**
- 若僅上傳圖片，系統會提示「尚未添加公關稿本體（非圖片附件）」。  
- 若附件過大，會自動改用 Drive 共享連結傳送。  
- 傳送中途失敗會提示「傳送失敗」，可重試。  

---

### 2) FB URL 流程（輸入 / 確認 / 傳送）

**目的**  
將 Facebook 分享連結傳送到目標信箱。

**前置條件**  
已開啟主界面，或已先傳送 FB URL 再 @Bot 喚起主界面。

**操作步驟**
1. 在主界面點擊「FB URL」。  
2. 按提示傳送包含 FB 分享連結的訊息。  
3. 若已先傳送 FB URL，可直接 @Bot 喚起主界面繼續操作。  
4. Bot 自動識別後，出現「✅ 傳送 FB URL / ✏️ 重新輸入」。  
5. 如需修改連結，點擊「✏️ 重新輸入」。  
6. 如需改動類型/語言，點「⚙️ 設定」。  
7. 確認無誤後點擊「✅ 傳送 FB URL」。  
8. 看到「會話結束」表示已完成。  

**注意事項**
- 僅支援 Facebook 相關連結（facebook.com / fb.com / fb.watch / fb.me）。  
- 若未偵測到連結，會提示「未偵測到 URL」。  

---

### 3) 附件刪除（單個 / 全部）

**目的**  
在傳送前調整附件列表，刪除不需要的檔案。

**前置條件**  
主界面「附件列表」裡已有檔案。

**操作步驟**
1. 在主界面點擊「刪除」。  
2. 看到每個附件前有「❌」按鈕。  
3. 點擊某個「❌ 檔名」進入刪除確認。  
4. 選擇「是，刪除」完成單個刪除。  
5. 需要全刪時點擊「🗑️ 全部刪除」。  
6. 在提示中確認「⚠️ 確認全部刪除」。  
7. 刪除完成後點擊「✅ 完成」返回主界面。  

**注意事項**
- 「全部刪除」會清空附件並結束會話。  
- 如果檔案已不存在，會提示「檔案不存在或已被刪除」。  

---

### 4) AI 模式設定（類型 / 優先度 / 語言 / 傳送方式）

**目的**  
在傳送前調整 AI 處理類型、優先度、語言與傳送方式。

**前置條件**  
已開啟主界面。

**操作步驟**
1. 點擊「⚙️ 設定」。  
2. 在列表中選擇：類型、優先度、語言、傳送方式。  
3. 目前選中項會顯示「✅」。  
4. 確認無誤點擊「確認」。  
5. 若不想儲存，點擊「取消」。  
6. 如已修改，系統會提示是否放棄更改。  
7. 返回主界面後查看設定是否已更新。  

**注意事項**
- 傳送方式為「Google Drive」時，附件將透過 Drive 共享連結傳送。  
- FB URL 的設定僅支援「類型 / 語言」，入口在 FB URL 選單內。  

---

### 5) UI 會話操作（刷新 / 結束 / 返回）

**目的**  
管理目前會話狀態與界面內容。

**前置條件**  
已開啟主界面或功能子選單。

**操作步驟**
1. 點擊「🔄 刷新」更新主界面。  
2. 點擊「🛑 結束會話」主動結束目前會話。  
3. 進入子選單後可用「⬅️ 返回主選單 / 返回」回到主界面。  
4. 若看到「新 UI 已生成」，請到最新訊息繼續操作。  
5. 若會話逾時，會提示「10分鐘無操作，會話自動結束」。  

**注意事項**
- 刷新會重新生成 UI，舊 UI 可能無法繼續操作。  

---

### 6) Logs 查看（篩選 / 關鍵字 / 翻頁 / 詳情）

**目的**  
查看近期傳送記錄、失敗原因與詳情。

**前置條件**  
已開啟主界面。

**操作步驟**
1. 點擊「🧾 Logs」。  
2. 選擇時間範圍：「1天 / 3天 / 7天」。  
3. 選擇狀態：「全部 / 成功 / 失敗」。  
4. 點擊「🔍 關鍵字」輸入關鍵字進行篩選。  
5. 點擊「❌ 清除」可清空關鍵字。  
6. 使用「⬅️ 上一頁 / ➡️ 下一頁」翻頁。  
7. 點擊列表中的記錄按鈕查看詳情。  
8. 點擊「🔄 刷新」拉取最新記錄。  
9. 點擊「⬅️ 返回」回到主界面。  

**注意事項**
- 初次進入或快取過期時會自動刷新。  
- 關鍵字匹配範圍是「標題 / Subject」。
