# 教学引导操作指南（todo）

本文件用于整理“帮助/教学引导”的操作说明，便于后续引导模式落地。按钮名称以当前 UI 为准，全部以繁体字样呈现。

---

## 1) 发送公关稿（附件上传 → 确认发送）

**目的**  
把公关稿正文与附件发送到目标邮箱，并完成本次会话。

**前置条件**  
已打开主界面，或已先发送附件后再 @Bot 唤起主界面。

**操作步骤（5–8 步）**  
1. 向 Bot 发送公关稿正文文件（非图片）与需要的附件/图片。  
2. 如尚未出现主界面，可在发送附件后 @Bot 唤起。  
3. 等待 Bot 提示“已添加…当下累计…”。  
4. 在主界面确认“附件列表”是否完整。  
5. 如需补充或删除附件，先完成附件调整。  
6. 点击“確認”开始发送。  
7. 观察进度提示，等待“傳送完成”。  
8. 若提示“會話結束”，表示本次已完成。

**常见问题/注意事项**  
- 若仅上传图片，系统会提示“尚未添加公关稿本体（非图片附件）”。  
- 若附件过大，会自动改用 Drive 共享链接发送。  
- 发送中途失败会提示“傳送失敗”，可重试。

## 2) FB URL 流程（输入/确认/发送）

**目的**  
将 Facebook 分享链接发送到目标邮箱。

**前置条件**  
已打开主界面，或已先发送 FB URL 再 @Bot 唤起主界面。

**操作步骤（5–8 步）**  
1. 在主界面点击“FB URL”。  
2. 按提示发送包含 FB 分享链接的消息。  
3. 若已先发送 FB URL，可直接 @Bot 唤起主界面继续操作。  
4. Bot 自动识别后，出现“✅ 傳送 FB URL / ✏️ 重新輸入”。  
5. 如需修改链接，点击“✏️ 重新輸入”。  
6. 如需改动类型/语言，点“⚙️ 設定”。  
7. 确认无误后点击“✅ 傳送 FB URL”。  
8. 看到“會話結束”表示已完成。

**常见问题/注意事项**  
- 仅支持 Facebook 相关链接（facebook.com / fb.com / fb.watch / fb.me）。  
- 若未检测到链接，会提示“未偵測到 URL”。  

---

## 3) 附件删除（单个/全部）

**目的**  
在发送前调整附件列表，删除不需要的文件。

**前置条件**  
主界面“附件列表”里已有文件。

**操作步骤（5–8 步）**  
1. 在主界面点击“刪除”。  
2. 看到每个附件前有“❌”按钮。  
3. 点击某个“❌ 文件名”进入删除确认。  
4. 选择“是，刪除”完成单个删除。  
5. 需要全删时点击“🗑️ 全部刪除”。  
6. 在提示中确认“⚠️ 確認全部刪除”。  
7. 删除完成后点击“✅ 完成”返回主界面。

**常见问题/注意事项**  
- “全部刪除”会清空附件并结束会话。  
- 如果文件已不存在，会提示“檔案不存在或已被刪除”。

## 4) AI 模式设置（类型/优先度/语言/传送方式）

**目的**  
在发送前调整 AI 处理类型、优先度、语言与传送方式。

**前置条件**  
已打开主界面。

**操作步骤（5–8 步）**  
1. 点击“⚙️ 設定”。  
2. 在列表中选择：类型、优先度、语言、传送方式。  
3. 当前选中项会显示“✅”。  
4. 确认无误点击“確認”。  
5. 若不想保存，点击“取消”。  
6. 如已修改，系统会提示是否放弃更改。  
7. 返回主界面后查看设置是否已更新。

**常见问题/注意事项**  
- 传送方式为“Google Drive”时，附件将通过 Drive 共享链接发送。  
- FB URL 的设置只支持“类型/语言”，入口在 FB URL 菜单内。  

---

## 5) UI 会话操作（刷新/结束/返回）

**目的**  
管理当前会话状态与界面内容。

**前置条件**  
已打开主界面或功能子菜单。

**操作步骤（5–8 步）**  
1. 点击“🔄 刷新”更新主界面。  
2. 点击“🛑 結束會話”主动结束当前会话。  
3. 进入子菜单后可用“⬅️ 返回主選單/返回”回到主界面。  
4. 若看到“新 UI 已生成”，请到最新消息继续操作。  
5. 若会话超时，会提示“10分鐘無操作，會話自動結束”。  

**常见问题/注意事项**  
- 刷新会重新生成 UI，旧 UI 可能无法继续操作。

## 6) Logs 查看（筛选/关键字/翻页/详情）

**目的**  
查看近期发送记录、失败原因与详情。

**前置条件**  
已打开主界面。

**操作步骤（5–8 步）**  
1. 点击“🧾 Logs”。  
2. 选择时间范围：“1天 / 3天 / 7天”。  
3. 选择状态：“全部 / 成功 / 失敗”。  
4. 点击“🔍 關鍵字”输入关键词进行筛选。  
5. 点击“❌ 清除”可清空关键词。  
6. 使用“⬅️ 上一頁 / ➡️ 下一頁”翻页。  
7. 点击列表中的记录按钮查看详情。  
8. 点击“🔄 刷新”拉取最新记录。  
9. 点击“⬅️ 返回”回到主界面。

**常见问题/注意事项**  
- 初次进入或缓存过期时会自动刷新。  
- 关键词匹配范围是“标题/Subject”。
